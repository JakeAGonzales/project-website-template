Hierarchical Decision Framework for Multi-Agent Path Finding

First Author, Second Author, Third Author
University of Washington1
University of New Mexico2
Paper Code arXiv

Abstract

Multi-agent path finding (MAPF) is a well-studied problem in robotics that aims to find collision-free paths for multiple agents in a shared environment. While various solvers exist, they all struggle with scalability as the size of the environment and number of agents increases, primarily due to the problem's combinatorial nature and the lack of consideration for congestion effects caused by agent interactions. To address these limitations, we introduce a novel hierarchical decision-making framework that integrates global flow optimization with local path planning. The framework first decomposes the environment into subregions, allowing it to manage complexity by planning paths for agents within localized subproblems. At the global level, we model the high-level coordination between subregions as a routing game, computing equilibrium flow distributions that guide agents' paths through the environment while balancing congestion between subregions. Additionally, we develop a deep learning model to predict future spatiotemporal congestion effects within the regions. These congestion predictions are utilized in the local planning, where the framework uses MAPF algorithms to compute collision-free paths that avoid high-congestion areas. By integrating global flow optimization with localized, congestion-aware planning, the proposed framework is able to compute high-quality solutions for large-scale MAPF problems. Our experimental results demonstrate improvements over traditional MAPF solvers in terms of success rate and computational time on large-scale problems.
